Movie UI Dashboard is a responsive React dashboard for exploring, filtering, and maintaining movie data through both REST and GraphQL APIs. It is built to make API behavior visible, compare transport layers, and manage movie records with a clean table-first interface.

## GitHub Description

React + TypeScript movie dashboard built with Vite, TanStack Query, TanStack Router, Axios, Redux Toolkit, Tailwind CSS, shadcn/ui, REST, and GraphQL.

## Highlights

- Switch between REST and GraphQL data sources from the same UI
- Search, sort, paginate, and filter movies by genre, release date, and revenue ranges
- Choose visible GraphQL fields to compare payload behavior
- Create, edit, and delete movie records from the dashboard
- Responsive layout with light and dark themes
