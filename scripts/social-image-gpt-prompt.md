# GPT Social Image Prompt

Use this prompt with ChatGPT image generation or any GPT-based image tool to create a GitHub social preview image for this repo.

## Prompt

Create a polished GitHub social image for a web app called **Movie UI Dashboard**.

Canvas size: **1280 x 640**.

Style direction:
- cinematic product marketing
- modern frontend dashboard aesthetic
- premium, sharp, editorial, clean
- dark interface with glowing highlights
- high contrast, crisp typography, subtle depth
- sophisticated, not cartoonish

The image should represent a movie data dashboard built with this tech stack:
- React 19
- Vite
- TypeScript
- TanStack Query
- TanStack Router
- Axios
- GraphQL
- REST APIs
- Tailwind CSS
- shadcn/ui
- Redux Toolkit
- ESLint
- Vitest

Visual composition:
- show a sleek analytics-style movie dashboard UI mockup
- include a table or card layout with movie titles, genres, dates, and revenue figures
- suggest both REST and GraphQL support through elegant labels, tabs, or data panels
- include subtle visual hints of filtering, sorting, and CRUD actions
- use a bold hero title: **Movie UI Dashboard**
- include a smaller supporting line: **REST + GraphQL movie data management built with React, TypeScript, and TanStack**
- add small refined stack chips for: Vite, TypeScript, TanStack Query, TanStack Router, Axios, GraphQL, REST, Tailwind CSS, shadcn/ui, Redux Toolkit

Color palette:
- charcoal, slate, and near-black base tones
- bright orange or amber cinematic accent
- supporting cyan and soft green for data highlights
- clean white typography

Typography:
- modern sans serif
- strong headline hierarchy
- readable at thumbnail size

Layout guidance:
- keep the title large and readable on the left or upper-left
- place the dashboard product mockup on the right or centered-right
- maintain generous spacing
- avoid clutter and tiny illegible labels
- make it feel like a real SaaS/movie analytics product

Avoid:
- generic AI art
- noisy backgrounds
- purple-heavy palettes
- overly futuristic holograms
- film reels as the main subject
- messy text
- low-contrast UI
- too many logos

Output requirements:
- export as a single clean social card image
- no watermark
- no browser chrome
- no device frame
- optimized for GitHub repository social preview

## Short Variant

Design a cinematic GitHub social image for **Movie UI Dashboard**, a modern React dashboard for browsing, filtering, and managing movie data with **REST and GraphQL**. Show a sleek dark analytics UI with movie tables, filters, revenue metrics, and clean product marketing typography. Include the title **Movie UI Dashboard** and a subtitle about React, TypeScript, TanStack Query, TanStack Router, Axios, Tailwind CSS, shadcn/ui, and Redux Toolkit.
